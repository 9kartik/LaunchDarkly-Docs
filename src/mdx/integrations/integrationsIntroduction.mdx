## Overview

This topic explains which integrations from third-party apps LaunchDarkly supports that let you
configure the product to your specific needs. You can also connect various destinations to receive
LaunchDarkly events.

## LaunchDarkly's integrations

LaunchDarkly supports the following integrations:

- Azure DevOps (formerly VSTS)

- Bitbucket Pipelines

- Git code references

  - GitHub Actions

  - Bitbucket Pipelines

  - CircleCI Orbs

  - GitLab CI

  - Custom configuration with the CLI
