## Overview

This topic explains how to generate a secure mode hash for various server-side SDKs.

## Using secure mode hashes

A secure mode hash lets the JavaScript SDK connect to a server-side SDK in secure mode.

Each SDK uses a unique method to create the hash.

This feature is available in the following SDKs:

* [.NET](#net)
* [Go](#go)
* [Java](#java)
* [Node.js (server-side)](#nodejs-server-side)
* [PHP](#php)
* [Python](#python)
* [Ruby](#ruby)

### .NET

The [`SecureModeHash`](http://launchdarkly.github.io/dotnet-server-sdk/html/M_LaunchDarkly_Client_LdClient_SecureModeHash.htm) method computes an HMAC signature of a user signed with the client's SDK key. If you use our [JavaScript SDK](/sdk/client-side/javascript) for client-side flags, this method generates the signature you need for secure mode.

Here is the method:

<CodeTabs
  defaultValue="csharp"
  values={[
    { label: 'Example', value: 'csharp', },
  ]
}>
<CodeTabItem value="csharp">

```csharp
var hash = ldClient.SecureModeHash(user);
```

</CodeTabItem>
</CodeTabs>
