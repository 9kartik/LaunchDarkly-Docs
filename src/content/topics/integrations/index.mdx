This category has documentation topics about integrations from third-party apps. LaunchDarkly supports these integrations to let you configure the product to your specific needs. You can also connect various destinations to receive LaunchDarkly events.

## Related content

Here are the topics in this category:

## LaunchDarkly integrations

LaunchDarkly supports the following integrations:

- [Azure DevOps (formerly VSTS)](/integrations/azure-devops)
- [Bitbucket Pipelines](/integrations/bitbucket-pipelines)
- [Jira](/integrations/jira)
- [Microsoft Teams](/integrations/microsoft-teams)
- [New Relic](/integrations/newrelic)
- [Slack](/integrations/slack)
- [Terraform](/integrations/terraform)
- [Visual Studio Code](/integrations/vscode)
- [Webhooks](/integrations/webhooks)

LaunchDarkly also offers a Git code references integration with support for the following tools:

- [Git code references](/integrations/git-code-references)

  - [GitHub Actions](/integrations/github-actions)
  - [Bitbucket Pipelines](/integrations/bitbucket-pipelines)
  - [CircleCI Orbs](/integrations/circleci-orbs)
  - [GitLab CI](/integrations/gitlab-ci)
  - [Custom configuration with the CLI](/integrations/custom-config)

## Data Export destinations

You can export events from LaunchDarkly to a third party data export destination. To learn more, read [Data Export](/integrations/data-export).

LaunchDarkly's Data Export feature supports the following destinations:

- [Google Cloud Pub/Sub](/integrations/google-cloud-pubsub)
- [Kinesis](/integrations/kinesis-destination)
- [mParticle](/integrations/mparticle-destination)
- [Segment](/integrations/segment-destination)
